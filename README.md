# Optional strength password

In a project, customer asked for optional rule for strength password which is not required but if user follow the optional rule, it effects on strength of password.

We have 4 rules:
1. At least 8 characters             ---> required
2. At least one uppercase letter     ---> required
3. At least one number               ---> reuqired
4. At least one spectial character   ---> optional

I use Form control + Angular Material in project
